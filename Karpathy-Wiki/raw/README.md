# /raw — the inbox (immutable)

Drop your **raw, unmodified source material** here:

- PDFs (papers, books, slides)
- transcripts (talks, podcasts, videos)
- articles, web clippings
- your own loose notes

## Rules

- **This folder is the source of truth and is treated as immutable.** Claude
  reads from here but never edits, renames, or deletes anything in it. That way
  your originals are always preserved exactly as you added them.
- Everything Claude produces from these files goes into `/wiki`, not here.

## How to use

1. Add one or more files to this folder.
2. Ask Claude: _"Read the raw/ folder, follow CLAUDE.md, and compile or update
   the /wiki folder with this new material."_
3. Check `/wiki/index.md` for the new/updated pages.

> Tip: give files descriptive names (e.g. `karpathy-2023-state-of-gpt.md`),
> because compiled wiki pages cite their origin file by name in their **Sources**
> section.
